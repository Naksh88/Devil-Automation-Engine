# 🔥 Devill AI Assistant

Devill AI Assistant is a Python-based voice-controlled desktop assistant that performs real-time tasks using natural speech commands. It intelligently greets users based on the time of day and enables hands-free control over applications, websites, and system operations, making everyday computing faster and more efficient.

---

## 🚀 Features

- 🎙️ Voice command recognition  
- ⏰ Time-based smart greeting  
- 🌐 Open websites (Google, ChatGPT, YouTube, etc.)  
- 💻 Launch desktop applications (Chrome, VS Code, Spotify, etc.)  
- 🕒 Tell current time  
- ❌ Close current or all applications  
- 🔌 System controls (Shutdown, Restart, Sleep)  
- 🗣️ Text-to-speech responses  

---

## 🛠️ Tech Stack

- Python  
- SpeechRecognition  
- pyttsx3  
- OS & Webbrowser modules  

---

## ⚙️ Installation

1. Clone the repository  
git clone https://github.com/your-username/devill-ai.git  
cd devill-ai  

2. Install dependencies  
pip install pyttsx3 SpeechRecognition pyaudio  

Note: PyAudio may require additional setup depending on your system.

---

## ▶️ Usage

Run the assistant  
python main.py  

Example voice commands:
- Open Chrome  
- Open YouTube  
- What is the time  
- Close all apps  
- Shutdown system  
- Exit
